If you are failing to connect to twitch after filling everything out, look at this:

On the twitch token page, double check that the following permissions are checked Yes (it randomly unchecks some):

  • bits:read
  • chat:read
  • channel:read:subscriptions
  • channel:read:redemptions
  • user:read:follows Having any of these settings unchecked will result in a Twitch Connection error
